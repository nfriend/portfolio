Beginning in 2013, DuPont Pioneer's Research IM department began a concerted effort to provide a consistent end-user experience across its entire application portfolio. The result of this effort was SoyLib, an opinionated WPF library that enforces a common backend structure and provides resuable frontend components.


Dates:
May 2013 - present
Technologies:
WPF, Microsoft Prism
How I Contributed:
Designed and developed a majority of the framework's visual components

Modals

The most important (and most complex) UI components provided by SoyLib are modals. Inspired by Twitter Bootstrap, SoyLib allows developers to easily show custom content as a modal, and gracefully handles displaying multiple modals.

Showing a modal is simple:

var uiManager = new UserInteractionManager(); uiManager.ShowCustomModal("MyCustomView"); // where MyCustomView has been registered with the DI container

In addition, SoyLib ships with some stock views that mimic WPF's built-in ShowMessageBox() functionality:

uiManager.ShowMessageBox("Your Message Box", "Here's a message box for you", MessageBoxButton.OK);
